# ickplayer

wordpress插件，无广告调用优酷，土豆，音乐台等视频网站，

调用方法为：
[ick] http://http://v.yinyuetai.com/video/644116 [/ick]

下载链接：
右边，看见没有？download zip,点击下载压缩包，直接上传wordpress插件目录启用即可。
